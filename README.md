# BOBOTexture V2

> 支持 Unity TextMeshPro 的图集打包工具 — 一键导出 TMP Sprite Asset

## 🚀 快速使用

### 1. 下载
从 [Releases](https://github.com/hontbei/BOBOTexture/releases) 下载最新 `BOBOTextureV2-v0.1.0.exe`，双击运行（无需安装）。

### 2. 添加精灵
- **拖拽** PNG/JPG 图片到左侧面板
- 或点击顶部 **Open** 按钮选择图片

### 3. 设置导出
在顶部导出栏：
- 勾选导出格式（**TMP Sprite Asset** 可直用于 Unity）
- 选择**输出目录**
- 输入**图集名称**

### 4. 打包
点击顶部的 **开始打包** 按钮即可。

## 🔧 功能详解

### 工具栏

| 按钮 | 快捷键 | 说明 |
|------|--------|------|
| **新建** | Ctrl+N | 清空所有精灵和设置 |
| **打开** | Ctrl+O | 选择图片或 .boboproj 项目文件 |
| **保存** | Ctrl+S | 保存当前项目到 .boboproj 文件 |
| **另存为** | Ctrl+Shift+S | 选择新路径保存项目 |
| **撤销** | Ctrl+Z | 撤销精灵增删操作 |
| **⚙ 设置** | — | 切换界面语言（中文 / English） |

### 导出栏

| 控件 | 说明 |
|------|------|
| **格式选择** | PNG Only / JSON Array / TMP Sprite Asset |
| **输出目录** | 打包文件的目标路径，点击浏览选择 |
| **图集名称** | 输出文件的基础名称，如 `atlas` → `atlas.png` |
| **开始打包** | 执行打包，完成后自动打开输出目录 |

### 左侧：精灵列表

| 操作 | 说明 |
|------|------|
| **拖拽图片** | 将图片拖入虚线框即可添加 |
| **点击虚线框** | 打开文件选择对话框 |
| **搜索框** | 按名称过滤精灵 |
| **清空** | 移除所有精灵 |
| **× 按钮** | 删除单个精灵 |
| **右键** | 弹出移除选项 |
| **Delete 键** | 删除选中的精灵 |

### 中间：预览区域

| 操作 | 说明 |
|------|------|
| **滚轮** | 缩放预览（10% ~ 400%） |
| **拖拽** | 平移预览 |
| **点击精灵** | 选中，左侧列表同步高亮 |
| **悬停精灵** | 预览中高亮对应区域 |
| **网格** | 显示/隐藏 32px 参考线 |

### 右侧：打包设置

| 设置 | 说明 |
|------|------|
| **布局** | 算法（MaxRects / Skyline）、自动尺寸、最大宽高 |
| **间距** | Shape Padding（精灵间距）、Border Padding（外边距）、Extrude（外扩） |
| **裁剪** | Trim 模式（None / Alpha）、Alpha 阈值 |
| **高级** | 允许旋转（TMP 格式自动禁用） |
| **结果** | 打包后显示统计信息、输出文件列表、TMP 用法示例 |

### 结果区

打包完成后显示：
- 📊 已打包 / 已跳过 / 输出文件数量
- 📁 输出文件列表
- 📋 TMP 用法示例：所有精灵的 `<sprite name="xxx">` 标签，可一键复制

## 📦 导出格式

| 格式 | 输出文件 | 用途 |
|------|----------|------|
| **PNG Only** | `名称.png` | 纯图集图片 |
| **JSON Array** | `名称.png` + `名称.json` | TexturePacker 兼容格式 |
| **TMP Sprite Asset** | `名称.png` + `名称.png.meta` + `名称.asset` + `名称.asset.meta` | Unity TextMeshPro |

TMP 导出会将图集 PNG 连同 Unity 所需的 `.meta` 和 `.asset` 文件一并生成，放入 Unity 项目的 Assets 目录后即可在 TMP 中使用 `<sprite name="xxx">` 标签渲染精灵。

## 🛠 技术栈

- **前端**：Vue 3 + TypeScript + Pinia + Vite
- **后端**：Rust（Tauri 2）
- **打包算法**：MaxRects（自实现）+ Skyline
- **导出**：Unity TMP Sprite Asset YAML 生成 + JSON Array

---

## 📝 项目文件

`.boboproj` — JSON 格式，保存所有精灵路径和打包参数。双击或 Ctrl+O 打开，Ctrl+S 保存。
